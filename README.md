# 🌱 Ecolution

Ecolution is a gamified sustainability platform that encourages eco-friendly habits through activity verification, rewards, and an integrated UMKM marketplace.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Prisma ORM
- Neon PostgreSQL
- Auth.js
- Cloudinary
- Mailgun
- Vercel
